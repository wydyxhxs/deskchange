# Changelog

## v1.0.1

- Fixed auto-start background launch so the main window no longer remains on the desktop after system startup
- Reduced startup flash when launching directly into the tray
- Fixed the close button tray fallback so hiding to the tray no longer causes an extra visible flicker
- Refined tray-only startup behavior to avoid unnecessary startup UI

## v1.0.0

- Initial public release
- Added configurable hotkeys for virtual desktops 1-4
- Added desktop create and delete actions in the main window
- Added startup registration and tray-based background behavior
- Added switch animation toggle
- Added portable package and setup package
