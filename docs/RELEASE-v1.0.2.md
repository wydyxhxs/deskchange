# DeskChange v1.0.2

This is a small UI polish release focused on the hotkey input fields.

DeskChange is a Windows 11 virtual desktop manager with a compact Chinese UI, configurable desktop hotkeys, startup options, desktop create/delete actions, and optional switch animation control.

![DeskChange Main Window](https://raw.githubusercontent.com/wydyxhxs/deskchange/main/docs/screenshots/main-window.png)

## What's improved

- Fixed vertical alignment for text inside the hotkey input boxes
- Reworked the hotkey input field rendering to keep labels visually centered
- Preserved the existing hotkey recording and clear interaction

## Downloads

- `DeskChange-Setup.exe`
  - Standard installer build

- `DeskChange-portable.zip`
  - Portable build
  - Extract and run directly

## Notes

- This release is a UI refinement update
- No settings migration or workflow changes are required
