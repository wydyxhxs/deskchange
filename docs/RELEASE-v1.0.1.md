# DeskChange v1.0.1

This is a focused stability update for tray startup behavior.

DeskChange is a Windows 11 virtual desktop manager with a compact Chinese UI, configurable desktop hotkeys, startup options, desktop create/delete actions, and optional switch animation control.

![DeskChange Main Window](https://raw.githubusercontent.com/wydyxhxs/deskchange/main/docs/screenshots/main-window.png)

## What's improved

- Fixed startup behavior when DeskChange is launched automatically with Windows
- The app now stays in the tray more reliably instead of leaving the main window on the desktop
- Reduced the visible white flash during hidden startup
- Fixed the flicker that could appear when closing the main window back to the tray

## Downloads

- `DeskChange-Setup.exe`
  - Standard installer build

- `DeskChange-portable.zip`
  - Portable build
  - Extract and run directly

## Notes

- This release focuses on startup and tray behavior
- No workflow changes are required for existing users
